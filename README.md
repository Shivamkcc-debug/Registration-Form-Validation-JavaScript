Real-Time Registration Form with Validation (JavaScript)

A modern Registration Form Web Application built using HTML, CSS, and JavaScript.
This project demonstrates real-time form validation, password strength checking, localStorage data persistence, and dynamic table rendering.
Real-time Name Validation
Email Format Validation
Phone Number Validation (10 digits)
Password Strength Meter
Show / Hide Password Toggle
LocalStorage Integration
Dynamic User Table Display
Responsive and Clean UI
Real-time Error Messages

How It Works
The user fills the registration form.
Each field is validated in real-time using JavaScript events
The password field checks strength using length conditions.
The Show/Hide icon toggles the password visibility.
When the form is submitted:
User data is stored in a submissions array.
The array is saved in localStorage.
Stored users are displayed dynamically in a table below the form.
